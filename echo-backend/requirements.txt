# ============================================================
# NOTES ON KNOWN ISSUES
# ============================================================
#
# 1. numpy on Python 3.14
#    pip may try to compile numpy from source and fail without
#    MSVC C++ Build Tools. Fix:
#      pip install numpy --only-binary=:all: --pre
#
# 2. PyAudio (DO NOT USE on Python 3.14)
#    No pre-built wheel exists for Python 3.14.
#    Use sounddevice instead — it is already listed above.
#
# 3. aws-sdk-bedrock-runtime conflicts with nova-act numpy pin
#    If nova-act tries to downgrade numpy and fails, install with:
#      pip install nova-act --no-deps
#      pip install fire install-playwright bedrock-agentcore Deprecated
#
# 4. strands-agents[bidi] pulls in PyAudio via BidiAudioIO
#    Do NOT install the [bidi] extra — use plain strands-agents
#    and the custom echo_sonic.py voice engine instead.
#
# 5. OpenSearch policies require 30s propagation after creation
#    Run setup_opensearch.py once after creating the collection.
#    The script handles the wait automatically.
#
